ReflexLab Account Deletion

Users can request deletion of their ReflexLab account and associated data by emailing:

[vamsikarnipu@gmail.com](mailto:vamsikarnipu@gmail.com)

Please include the email address associated with your account.

Upon receiving a request, we will permanently delete:

* Account information
* Profile data
* Game progress
* Statistics and achievements
* Associated cloud data

Deletion requests are processed within 30 days.

If you are using a guest account that has not been linked to an email address, uninstalling the application will remove local device data.
